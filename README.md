# neural-network
a NEAT style (i think) neural network written from scratch in typescript

## installing/running
prereqs: nodejs, npm
1. clone the repo
```shell
git clone https://github.com/metalcupcake5/neural-network.git
```
2. install packages
```shell
cd neural-network
npm install
```
3. run
```sh
npm run run
```
